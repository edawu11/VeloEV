# veloev (vev)

Evaluation and visualization for RNA velocity methods.

## Installation

You can install `veloev` by cloning the repository and installing it via pip.

```bash
# 1. Clone the repository
git clone [https://github.com/edawu11/VeloEV.git](https://github.com/edawu11/VeloEV.git)

# 2. Navigate into the directory
cd veloev

# 3. Install the package
# Option A: For normal use
pip install .

# Option B: For development (Recommended)
# Use this if you plan to modify the code. Changes will take effect immediately.
# pip install -e .
